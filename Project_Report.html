01Abstract

Money Mitra is a full-stack digital banking simulation platform built with React.js, Node.js/Express, and MySQL — replicating the core features of a real-world neobank including multi-step KYC, UPI fund transfers, virtual cards, loan management, bill payments, support ticketing, and a comprehensive admin panel.

The platform implements industry-standard security: JWT refresh tokens, bcrypt hashing, rate limiting, SQL parameterization, and fraud detection algorithms. Its premium dark-mode glassmorphism UI features smooth animations and full mobile responsiveness.

07System Modules

Landing Page

Standalone animated HTML with glassmorphism hero, CSS keyframe animations (blob, card float, ring spin), IntersectionObserver scroll-reveal, animated counters, features grid, testimonials, FAQ accordion, and smart file:// detection.

Authentication

4-step smart registration form (Identity → Account → Contact → Security). Auto-advances steps, validates age (18+), grants ₹10,000 welcome bonus. JWT access token (15min) + refresh token (7 days, stored in DB). Stale auth prevention on frontend.

Core Banking Modules

/api/transaction

💸 Fund Transfer

IMPS/NEFT via account + IFSC. Beneficiary management, balance validation, atomic MySQL transactions, reference numbers, live history.

/api/upi

⚡ UPI Module

@moneymitra UPI ID. Send/receive via UPI ID, ₹2L limit, zero fees, instant lookup, QR simulation.

/api/cards

💳 Cards Module

Virtual debit card issuance. Freeze/unfreeze, spending limit config, multi-card support.

/api/bills

🧾 Bill Payments

Electricity, Water, Gas, DTH, Mobile, Internet. Balance deduction, receipt generation, history.

/api/loans

🏦 Loans

Personal (12.5% p.a.) & Home (8.5% p.a.). EMI calculator, repayment schedule, admin disbursement.

/api/kyc

🪪 KYC System

PAN + Aadhaar submission. Admin approval auto-generates immutable CKYC number (permanently locked in DB).

/api/support

🎧 Support

Ticket creation, status tracking, admin response, platform FAQ with accordion search.

/api/notifications

🔔 Notifications

Bell icon with unread badge. Transaction alerts, KYC updates, loan events. Mark read / clear all.

10Security Features

🔑
Password Hashing
bcryptjs — 12 salt rounds, ~300ms/hash — computationally expensive to brute-force
🎟️
JWT Authentication
Access token: 15min expiry. Refresh token: 7 days, stored hashed in DB
🔄
Token Rotation
Refresh tokens revoked immediately on logout — prevents token reuse
🚦
Rate Limiting
200 req/15min global · 10 login attempts/hour per IP
🛡️
Helmet.js
Sets 14 HTTP security headers (CSP, HSTS, X-Frame-Options, etc.)
🌐
CORS Policy
Restricted to http://localhost:5173 only in development
💉
SQL Injection Prevention
All DB queries use ? parameterized placeholders via mysql2
✅
Joi Schema Validation
Every API endpoint validates request body against a defined schema
🚨
Fraud Detection
Auto-flags transactions >₹50,000 or rapid successive transfers
🔒
CKYC Immutability
ckyc_locked = true after KYC approval — permanently immutable

14Limitations

⚠️
Simulation NoticeThe following exist due to the academic/educational nature of this project and do not affect the core demonstration of software engineering concepts.
  1. Simulation Only — No real payment gateways (Razorpay, PayU) or RBI connectivity
  2. No OTP — Phone/email verification does not send real SMS/email OTPs
  3. No 2FA — TOTP concept shown in UI but not fully implemented
  4. Local Only — Requires local MySQL; not deployed to cloud
  5. No WebSockets — Real-time updates use polling (10-second intervals)
  6. Single Currency — Indian Rupees (₹) only; no multi-currency
  7. Rule-Based Fraud — Threshold-based detection, not ML/AI anomaly detection
  8. Dev Mode — Vite dev server only; no Nginx/PM2 production setup

16Conclusion

Money Mitra successfully demonstrates the development of a production-grade full-stack web application simulating a complete digital banking experience. The project integrates:

  • Animated landing page as the platform's public interface
  • Secure React SPA with 12 functional pages covering the full user banking journey
  • Robust Node.js/Express REST API — 93 endpoints across 12 modules
  • MySQL relational database with 11 tables, proper integrity & ACID transactions
  • Industry-standard security: JWT, bcrypt, rate limiting, SQL parameterization, fraud detection
  • Comprehensive admin panel enabling full platform oversight and management
  • Live transaction monitoring with direction-aware Debit/Credit display
🏆
Project OutcomeMoney Mitra meets all defined primary and secondary objectives, demonstrating comprehensive command of modern full-stack development — a strong foundation for future enhancements including real payment gateway integration and cloud deployment.
